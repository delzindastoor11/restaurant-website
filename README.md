# Donut Whole – Vegan Donut Shop Website

A responsive restaurant website for **Donut Whole**, a vegan donut shop based in Mumbai.  
This project demonstrates a modern frontend layout built with structured HTML, advanced CSS styling, and basic JavaScript for interactivity.

---

## Live Demo
https://delzindastoor11.github.io/restaurant-website/

---

## Features
- Clean and responsive layout
- Modern navigation bar
- Menu section with donut items and pricing
- Image-based product showcase
- Semantic HTML structure
- Organized asset management (CSS, JavaScript, Images)

---

## Tech Stack
- **HTML5** – semantic markup and structure  
- **CSS3** – layouts, styling, and responsiveness  
- **JavaScript** – basic interactivity  
- **GitHub Pages** – deployment

---

## Purpose
This project was created to practice frontend web development concepts, including layout design, responsive styling, asset organization, and static site deployment using GitHub Pages.
